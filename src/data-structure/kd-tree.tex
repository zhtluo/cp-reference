Find the $k$-th closest/farthest point in $O(kn^{1-\frac{1}{k}})$.

Usage:
\begin{compactenum}
\item Store the data in \mintinline{cpp}|p[]|.
\item Execute \mintinline{cpp}|init|.
\item Execute \mintinline{cpp}|min_kth| or \mintinline{cpp}|max_kth| for queries (k is 1-based).
\end{compactenum}

Note: Switch to the commented code for Manhattan distance.
