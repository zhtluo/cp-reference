\begin{compactenum}

\item \mintinline{cpp}|rk[i]|: The rank of the suffix beginning at position $i$.
\item \mintinline{cpp}|height[i]|: The longest common prefix of \mintinline{cpp}|sa[i]| and \mintinline{cpp}|sa[i - 1]|.
